# Help- Logs,Properties,About DAC

In the Help menu item you can access to :

* Dac On -Line Documentation
* Logs&#x20;
* Properties&#x20;
* Abaut DAC

### Dac On -Line Documentation

To view the documentation  select the menu item **Help >> Dac On-Line Documentation**

### Logs

To view the logs file, select the menu item Help >>Logs. The list of all log files generated by the system will open.

In the log file , are recorded all the operations, in chronological order, carried out in the normal use of the DAC.

![](<../../../.gitbook/assets/image (944).png>)



If you do not have the possibility to download the Log from the Design Studio, you can download it directly from the Log folder in the Design Studio installer folder.

### Properties&#x20;

Select the menu item **Help >>Properties** to consult the file properties.

![](<../../../.gitbook/assets/image (1214).png>)

**builld.info:** you can view the information for example on the installed version of the studio design.

**decisyon.properties:** you can view the decisyon properties

**settings.properties:**  you can view the Decisyon settings properties&#x20;

### About DAC

Select the menu item **Help >>About DAC the information about the DAc Version** copyright and license.
