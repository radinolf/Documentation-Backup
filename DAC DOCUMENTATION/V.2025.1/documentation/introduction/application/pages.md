# Pages

## **Introduction**

**Pages** is a service developed and optimized specifically for business users, providing an intuitive interface to interact seamlessly with different business applications built with App Composer and Design Studio.&#x20;

Unlike AppComposer, created for app developers, Pages empowers business users by offering essential features and functionalities to use the business applications.

<figure><img src="../../../.gitbook/assets/image (2192).png" alt=""><figcaption></figcaption></figure>

Pages include the following main functionalities:

* [**Home Page and Navigation**](pages.md#home-page-and-navigation)**:** Pages interface allows business users to easily navigate through applications and pages, facilitating efficient content exploration.
* [**Profile Section**](../../user-management/user-profile-in-pages.md)**:** Users can customize their profiles and meet individual preferences using the **User** **Profile** interface, access and review the Terms of Use and Privacy Policy, and disconnect from the Pages application.
* [**User Management**](../../user-management/user-administration/)**:** Includes features for managing user accounts, organizations and editing settings.

## Home Page and Navigation

The Home Page and Navigation panel within Pages provide a structured and intuitive user experience:

### Main Toolbar

The **Main Toolbar** offers quick access to the main features:

<figure><img src="../../../.gitbook/assets/image (2200).png" alt=""><figcaption></figcaption></figure>

* **Pages**<img src="../../../.gitbook/assets/image (2195).png" alt="" data-size="line">**:** Navigate through Applications and the relative pages and contents.
* [**User Management**](../../user-management/)<img src="../../../.gitbook/assets/image (2196).png" alt="" data-size="line">: Access and manage the user's account and permissions. It is visible only to user with specific scopes.
*   **User Picture:** Clicking on the User picture provides access to the following sections:

    * [**User Profile Page**](../../user-management/user-profile-in-pages.md)**:** View detailed personal information, manage your profile picture, update settings related to location and default preferences, and security settings such as API token management.
    * [**Terms of Use/Privacy Policy**](../../app-functionality/terms-of-use-privacy-policy.md)**:** Review and accept the Terms of Use and Privacy Policy, verifying that you understand and agree to the legal obligations and rights associated with using the application. The terms of use file must be previsouly uploaded by an App Developer in App Composer service.
    * **Logout:** Securely quit and disconnect from the Pages application, ensuring your session is correctly terminated.

### Navigation Panel

The **Navigation Panel** enhances Applications and relative content navigation with the following elements:

<figure><img src="../../../.gitbook/assets/image (2197).png" alt=""><figcaption></figcaption></figure>

* **Search:** Quickly find pages within the application, even when folders are closed. Matches automatically expand corresponding folders upon detection.
* **Current Application:** Displays the Application currently in use and allows the user to select a different one.
* **Pages of the Application:** Lists the pages within the current application, facilitating easy navigation.
