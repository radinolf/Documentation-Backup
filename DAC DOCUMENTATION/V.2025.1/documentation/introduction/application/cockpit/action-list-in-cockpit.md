# Action List in Cockpit

The Action List section of Cockpit allows you to monitor the Action List scheduled in App Composer

<figure><img src="../../../../.gitbook/assets/image (2136).png" alt=""><figcaption></figcaption></figure>

From the top menu "Application," you can select the application for which you want to view the Action List. Once selected, the Action List and the associated Schedule Action List will be visible.

<figure><img src="../../../../.gitbook/assets/image (2137).png" alt=""><figcaption></figcaption></figure>

**Statistics:** It's possible to display the execution statistics generated by the Action List or by a single action.

* **Average duration** : average duration of the historical execution of the action list
* **Total execution count**: total number of executions (both successfull and in error)
* **Total execution error count**: total number of executions with status "error".

<figure><img src="../../../../.gitbook/assets/image (2138).png" alt=""><figcaption></figcaption></figure>

When selecting a **Schedule**, you can view the details.
