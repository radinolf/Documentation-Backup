# Advanced properties



**Export**&#x20;

* **zip-export-file** if enable all report exported will be compressed in zip format
* **disable-export-limit** if enabled it doesn't check report rows limit during the export. Exporting could take more time.

**Filters**&#x20;

* **disable-all-security-filters:** Disable the use of all Security Filter
* **disable-security-filters:** Sets the list of Security Filters to exclude

**SQL-custom**

* **having-clause:** Sets a custom HAVING condition, added to the report read query
