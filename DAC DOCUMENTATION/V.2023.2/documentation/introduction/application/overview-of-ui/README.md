# Overview of UI

Once logged in using the URL generated by the developer in DAC-DS, the DAC _Home Page_ opens

If you have one or more apps instantiated in your space.

The DAC interface consists of the following components:

* [**Toolbar**](./#main-toolbar): at the top of the interface and the shortcuts are available to access the user-related features associated with the DAC.
* [**Side Navigation Menu**:](./#side-navigation-menu) the button on the top left, You can access the commands for DAC's main features.

![](<../../../../.gitbook/assets/image (1404).png>)



In the Home Page you can see all applications associated with your group. If just one application has been associated with a user he will directly access the pages of the app after login.

![](<../../../../.gitbook/assets/image (1446).png>)

## Side Navigation Menu

It is possible to expand the main function area. Once expanded the main functions available are:

![](<../../../../.gitbook/assets/image (1477).png>)

* &#x20;**Home**, to return to the Home Page; you can also click on the Decisyon logo
* &#x20;**Create,** which opens a menu for creating new objects in DAC:&#x20;
  * [**Report**](../../../report/report/)**,**   to create a report  using the report editor
  * [**Document**](../../../collaboration/document/)**,**  to create a documents using an advanced text editor.
  * [**Tasks**](../../../collaboration/task.md)**,** to create a task that are not linked to analysis information
* [**Alerts,**](../../../app-functionality/alert-and-rules/alert.md) to create a new alert
* [**My Documents**](../../../collaboration/document/my-document/)**,** open the list of the documents created by the user logged &#x20;
*   **Online Help, opens the list of DAC documents**

    * DAC Manual
    * Rest API Documentation
    * Widget Designer Manual


* [**Widget Designer**](/broken/pages/-MfHjAsXURrVlxStmf2r)**, opens** the  development environment that provides a feature-rich toolset to build complex applications with rich user interfaces.
* **Administration**
  * [**User Administration,**](../../../user-management/users-administration.md) create or  modify the DAC users
  * [**Group administration**](../../../user-management/groups-administration.md)**,** create or modify the groups in DAC
  * [**User attributes,**](../../../user-management/user-attribute.md) create or modify the user attribute
  * [**Languages Administration**](../../../instance-configuration/multilanguage-and-localization/languages-administrations.md)**,** allows you to import new languages in addition to the default ones within the application.
  * [**Labels Management**](../../../instance-configuration/multilanguage-and-localization/labels-management.md)**,** mange the Dac labels to support the multilanguage
  * [**Custom error**](../../../instance-configuration/custom-error.md)**,** customize the error messages displayed by the user while using the application.
  * [**Jobs**](/broken/pages/-Mi0SPs4oskUYkDyLWIY)**,** list of Jobs imported into the DAC
  * [**Rest Api Client**](../../../sdk-and-api/rest-api-client.md)**,** opens to configure a call by a third party endpoint to take advantage of the services within the DAC
  * [**Application Synchronization**](../../../application-synchronization-wip/)**,** opens the interface to synchronize two applications
*   **Settings**

    * [**Logo & Colors**](/broken/pages/-MfHkt6_JwQYBalmnCGC)**,** opens the tools for customizing the DAC layout
    * [**General Information,**  ](/broken/pages/Z11vnQKECZlPpKVc4o7e)opens the panel that shows some summary information of the installed DAC.
