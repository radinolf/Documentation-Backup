# Other information

In the Help menu item you can access to :

* Dac On -Line Documentation
* Logs&#x20;
* Properties&#x20;
* About DAC

## Documentation

To view the documentation  select the menu item **Help >> Dac On-Line Documentation**

## Logs

To view the logs file, select the menu item Help >>Logs. The list of all log files generated by the system will open.

In the log file , are recorded all the operations, in chronological order, carried out in the normal use of the DAC.

You can find more information in the [dedicated page](../../instance-configuration/logs.md).

![](<../../../.gitbook/assets/image (1203).png>)



If you do not have the possibility to download the Log from the Design Studio, you can download it directly from the Log folder in the Design Studio installer folder.

## Properties&#x20;

Select the menu item **Help >>Properties** to open the file properties.

![](<../../../.gitbook/assets/image (1473).png>)



* **build.info:** contains detailed information about the version and the build of Design Studio
* **decisyon.properties:**  contains advanced properties of Design Studio
* **settings.properties:**  contains advanced settings of Design Studio

{% hint style="danger" %}
It is highly advisable not to modify these files unless you have a precise understanding of the changes you are making
{% endhint %}

## Increase the memory allocated to Design Studio

If necessary, you can modify the allocated Java memory for Design Studio by adjusting the configuration values. To do this, locate and open the file named `Decisyon App Composer Design Studio.vmoptions` in the installation folder. Once opened, change the following values accoringly to the memory available in your client:

```
-Xms1024m
-Xmx2048m
```
