# Column definition



&#x20;It is possible defining and customize columns on the **Report Editor** ![](<../../../.gitbook/assets/image (1816).png>)

This configuration is usually used to define the columns of the [Smart Grid](https://widgets.decisyon.com/widgets/#smart-grid) widget that inherits the associated report settings.

The[ **Columns-definition**](https://widgets.decisyon.com/smart-grid/documentation/smart-grid-9.1.1/report-model#columns-definition) properties allows you to define the report column settings. You can manipulate individual column properties.
