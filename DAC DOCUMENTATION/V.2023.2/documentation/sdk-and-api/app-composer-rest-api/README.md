## Introduction

The REST API allows developers to integrate their application with App Composer to read data, get configurations, create objects, modify settings etc.&#x20;

This documentation is intended as an essential resource for developers seeking information on how to interface with the API exposed by App Composer. If you have any questions or run into any issues while using App Composer API, donâ€™t hesitate to reach out your technical account or Decisyon's support.

### API Endpoint <a href="#api-endpoint" id="api-endpoint"></a>

The DAC API can be accessed appending the `/api/v../rest/` to the instance's hostname:

&#x20;`https://<dcy_url>/api/v1/rest/`&#x20;

&#x20;`https://<dcy_url>/api/v2/rest/`&#x20;

The API are grouped in two version: V1 and V2.&#x20;

For more details please see the documentation [**Swagger e Postman**](./#swagger-and-postman)**.**
