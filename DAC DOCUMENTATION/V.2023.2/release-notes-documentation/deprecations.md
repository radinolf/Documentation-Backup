# ğŸ”„ Deprecations

As App Composer technology evolves, some functions and features become obsolete or are replaced by more efficient alternatives. This page lists **deprecated functions** and, where possible, recommended replacements. Updating your applications to utilize these alternatives ensures continued performance and compatibility with future versions.

{% hint style="info" %}
**Note**: Applications relying on deprecated features may require modifications to remain operational.
{% endhint %}

We use the term "deprecation" to refer to the process of retiring a feature. When we announce that a feature is being deprecated, it immediately becomes deprecated. Some of these features might already have a removal date planned, while others may not, in which case their removal date is marked as "To be defined."

Typically, a feature is removed after two major release versions following its deprecation.

In example, if a feature is marked as deprecated in version 2023.0.0, it is guaranteed to remain available up through version 2023.2.0, including all minor updates (2023.2.x). However, starting with version 2023.3.0 and later, Decisyon Inc. may remove the deprecated features.

For each feature it is reported:

* **Feature Name:** This field indicates the name of the deprecated function or feature.
* **Impact:** This field provides a detailed description of the effects or impacts that will occur after the removal of the deprecated function. This may include information on how the removal will affect user workflows, system performance, or compatibility with other features.
* **Alternative Configuration:**  This field provides an alternative solution or new methodology that can be used in place of the deprecated function is indicated.&#x20;
* **Deprecation Date:** This field indicates the date and the DAC version when the feature was officially deprecated in the DAC .&#x20;
* **Estimated Removal Date:** This field shows the expected date for the complete removal of the deprecated function from the DAC. It helps in planning effectively for applications migration and updates. The icon ğŸ†• shows removal date that have now been declared.

{% hint style="danger" %}
In case the removal will affect features already available in a production environment please contact your account in Decisyon.
{% endhint %}
