# Migration Notes 6.3

Decisyon App Composer “Jalapeño” can be installed on new environment or, in case of updating existing Decisyon App Composer, on environment having installed Decisyon App Composer “Jake”. Any other previous version (e.g. “Matilda”) requires being update to “Jake” before installing “Jalapeño”.<br>
