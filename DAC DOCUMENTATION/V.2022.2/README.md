## Change Log

The Change Log' is a document distributed together with the release of a new version (minor release or patch) of the Decisyon App Composer Platform and is prepared as a log of solved bugs and of the functions entered in the Platform.



## Migration Notes

The purpose of this document is to show the changes and impact in migrating Decisyon App Composer projects. During the migration phase, the system administrator must consider the information reported in this document, in order to correctly perform the version passage. In the version passages, which mean a minor release change (e.g.: from Name1 to Name2), we strongly advise carrying out a check of the correct operation of the migrated applications in a test environment, prior to their actual passage to production.
