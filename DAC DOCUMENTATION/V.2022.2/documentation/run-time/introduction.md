# Introduction

DAC is the user interface of DAC’s Design Studio. The DAC environment is a web-based analysis where apps created in DAC Design Studio are published. To access the application need to enter the credentials provided by the administrator.

The analyses are carried out on reports and pages, that are collected in applications. The main DAC function is to facilitate collaborative analysis of the data, reports and KPIs available.

![](<../../.gitbook/assets/image (473).png>)



Once logged in using the URL generated by the developer in DAC-DS, the DAC _Home Page_ opens

If you have one or more apps instantiated in your space.

The DAC interface consists of the following components:

* [**Toolbar**](introduction.md#main-toolbar): at the top of the interface and the shortcuts are available to access the user-related features associated with the DAC.
* [**Side Navigation Menu**:](introduction.md#side-navigation-menu) the button on the top left, You can access the commands for DAC's main features.

![](<../../.gitbook/assets/image (358).png>)



In the Home Page you can see all applications associated with your group. If just one application has been associated with a user he will directly access the pages of the app after login.

![](<../../.gitbook/assets/image (617).png>)

## Side Navigation Menu

It is possible to expand the main function area. Once expanded the main functions available are:

![](<../../.gitbook/assets/image (245).png>)

* &#x20;**Home**, to return to the Home Page; you can also click on the Decisyon logo
* &#x20;**Create,** which opens a menu for creating new objects in DAC:&#x20;
  * [**Report**](../design-studio/application/report-management/report.md)**,**   to create a report  using the report editor
  * [**Document**](document.md)**,**  to create a documents using an advanced text editor.
  * [**Tasks**](task.md)**,** to create a task that are not linked to analysis information
  * [**Send Message**](sand-a-message.md), to create a message to be sent to one or more users.
* &#x20;[**Recent Contents**](http://documents.decisyon.com/tomcat/manuals/%7B%24%7DuwQw4waFztxG1nK5fKjSvwW6RZK2p9REodEaIslrXTY%3D/user/ge/en/web/recentContents.htm#_Recent_Contents) ,opens a list of the last contents viewed by the user.
* [**People**](people.md), opens the list of users belonging to the application.
* [**Alerts,**](alert.md) to create a new alert
* [**My Documents**](my-document/)**,** open the list of the documents created by the user logged &#x20;
*   **Online Help, opens the list of DAC documents**

    * DAC Manual
    * Rest API Documentation
    * Widget Designer Manual


* [**Widget Designer**](/broken/pages/-MfHjAsXURrVlxStmf2r)**, opens** the  development environment that provides a feature-rich toolset to build complex applications with rich user interfaces.
* **Administration**
  * [**User Administration,**](administration/users-administration.md) create or  modify the DAC users
  * [**Group administration**](administration/groups-administration.md)**,** create or modify the groups in DAC
  * [**User attributes,**](administration/user-attribute.md) create or modify the user attribute
  * [**Languages Administration**](administration/languages-administrations.md)**,** allows you to import new languages in addition to the default ones within the application.
  * [**Labels Management**](administration/labels-management.md)**,** mange the Dac labels to support the multilanguage
  * [**Custom error**](administration/custom-error.md)**,** customize the error messages displayed by the user while using the application.
  * [**Jobs**](/broken/pages/-Mi0SPs4oskUYkDyLWIY)**,** list of Jobs imported into the DAC
  * [**Rest Api Client**](administration/rest-api-client.md)**,** opens to configure a call by a third party endpoint to take advantage of the services within the DAC
  * [**Application Synchronization**](administration/application-synchronization-wip.md)**,** opens the interface to synchronize two applications
*   **Settings**

    * [**Logo & Colors**](settings/logo-and-colors.md)**,** opens the tools for customizing the DAC layout
    * [**General Information,**  ](settings/general-information.md)opens the panel that shows some summary information of the installed DAC.
