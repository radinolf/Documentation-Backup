# New Feature

### Scope

In this new version, **Scopes** have been introduced for Pages users. In addition to the standard user scopes, it is now possible to assign the '**Theme administrator**' Scope, which grants users the privilege to customize the theme of the application they belong to.

### Application Settings&#x20;

A new section "**Application settings**" has been added to the user profile panel for business users with the '**Theme Administrator**' role. This section allows them to manage application settings, including the ability to configure the application's theme directly from their profile."
