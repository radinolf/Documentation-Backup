# Know Issue

**Eclipse Path Retention After DAC Upgrade**

When Eclipse is already installed in Design Studio, and an upgrade to a newer version of DAC is performed, Eclipse continues to reference the path of the previous installation. This can cause conflicts or unexpected behavior when using Eclipse after the upgrade.

### Static Resources Not Properly Invalidated in Browser Cache After DAC Version Update

After upgrading to DAC version 2024.1.1, some users may not see the new features due to resources stored in the browser's cache. It is recommended to clear the browser cache to properly load the updates.
