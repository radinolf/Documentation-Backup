# Removals

### Incoming Mail

The **Incoming Mail** section have been  removed from the DAC.

<figure><img src="../../../.gitbook/assets/image (1952).png" alt=""><figcaption></figcaption></figure>

### People

The **People** section have been  removed from the DAC.

<figure><img src="../../../.gitbook/assets/image (1953).png" alt=""><figcaption></figcaption></figure>

### Recents Content

The **Recents Content** section is no longer available in the DAC.

<figure><img src="../../../.gitbook/assets/image (1921).png" alt=""><figcaption></figcaption></figure>

### Notification Designer&#x20;

The following type of notification attachment is no longer available in the Notification Designer

* WEB format as pages attachment
* PDF format as pages attachment
* WEB format as report attachment

<figure><img src="../../../.gitbook/assets/image (1914).png" alt=""><figcaption></figcaption></figure>

### Java Custom Total

**Java Custom Total** is no longer used in the Report.

<figure><img src="../../../.gitbook/assets/image (1917).png" alt=""><figcaption></figcaption></figure>

## Removal of Code Snippet templates

**Code Snippet tempalte** is no longer used in the Report.

<figure><img src="../../../.gitbook/assets/image (1918).png" alt=""><figcaption></figcaption></figure>

### Send a Message

In the User Profile is no longer available to use the "**Send Message**" component.

<figure><img src="../../../.gitbook/assets/image (1957).png" alt=""><figcaption></figcaption></figure>

### Legacy widget "Styled Selector"&#x20;

It will no longer be possible to use the legacy widget "Styled Selector" in the page designer.

Any installations that use the Style Selector on the pages will not be able to migrate by continuing to use this widget.



### Removal of properties join-global-name and level-global-name

The properties are `join-global-name` in the report and `level-global-names` in the levels have been removed.
