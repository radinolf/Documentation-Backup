# Install

## Pre-requisite

The pre-requisite to install Design Sutdio is the Java Runtime Environment 1.8 (JRE 1.8) or above.

During the installation process, if the JDK version required by Design Studio is not available in the client then the proper version is downloaded from internet and included in the installation folder of Design Studio

## Overview <a href="#overview" id="overview"></a>

**DAC Design Studio (DS).** DAC-DS is a powerful feature-rich environment for the high-speed development, configuration and administration of data-intensive IoT business applications.

You can download the Design Studio installer from [RunTime](https://documentation.decisyon.com/documentation/run-time/settings/general-information).. You need to have an active user account in DAC with “administrator” grant in order to download the installer. If you don’t have it, please ask to your system administrator.

To download and install the correct version of the Design Studio it’s possible directly from.

![](<../../../.gitbook/assets/image (894).png>)

The installation of the design studio is performed through a wizard. The installer can be executed on Windows. The installer automatically deploy the required OpenJDK to run Design Studio, ignoring any other java version already used by the client. When Design Studio is installed, each version will have its own folder in the O.S.

## How To Install Design Studio <a href="#how-to-install-design-studio" id="how-to-install-design-studio"></a>

Below is the step by step guide to create a new connection.

### 1. Connect on Runtime <a href="#id-1.-connect-on-runtime" id="id-1.-connect-on-runtime"></a>

The first step to install the Design Stusio is log in on Runtime with a user account in DAC with “administrator” grant.

![](<../../../.gitbook/assets/image (1244).png>)

### 2. Download Design Studio <a href="#id-2.-download-design-studio" id="id-2.-download-design-studio"></a>

Form the vertical toolbar select Settings>>General Information >> Activation Key section.

&#x20;

![](<../../../.gitbook/assets/image (1227).png>)

In the Activation Key section, select DONLOAD button for the item “Download Design Studio Installer for window”.

### 3. Installation Wizard

Once the package has been downloaded, the installation wizard starts. If a version of the Design Studio has already been installed on your device, the system will ask you whether to update it or install it in a different directory.

![](<../../../.gitbook/assets/image (1639).png>)

After the installation of the Design Studio the icon to start will be present on your device.

![](<../../../.gitbook/assets/image (1349).png>)

Now you can caccess on Design Studio. When you access on Design Studio for the first time, you need to create a New connection. Please see the documentation [Connection on Design Studio](/broken/pages/-MfHlVz4Q9wdh91vXMgG)

During the installation phase the folder proposed by the system is (c: / Progarm File…) but you must have the administrator grants (writing and creating files). So if you don't have the right to write to that folder, an exception is thrown. In this case it is recommended to select a folder with write grants for all Users that will run the DS.
