# Page Widget



**Widgets** are the fundamental bricks of a page and provide specific functionalities that you can leverage to build your page. Each widget has its own configuration properties; a page contains one or more widgets and you can also create interactions on them and they can leverage the same data.

Main widgets groups that can be used in a page are:

* [Data Visualization](data-visualization.md): set of widgets to visualize the data
* [Form](form.md): set of widgets used to insert a text or select page by
* [Button](button.md): set of widgets used to perform the operations which interact with other systems or other DAC objects
* Containers: set of widgets used to define a Page layout in a more complex manner than the simple grid available on the Page Designer
* [Discussion](discussion.md): set of widgets use for discussions to be opened, possibly on a specific topic
* [Resource](resource.md)**:** set of widgets to insert in the page image JavaScript or text
* [Social Spaces](activity-stream.md): set of widgets used in the social space.
