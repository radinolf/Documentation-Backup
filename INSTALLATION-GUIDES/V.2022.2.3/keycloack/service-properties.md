# Service properties

| Property                   | Description                                                                                                                                                                                          | Type              | Mandatory | Default Value |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------- | --------- | ------------- |
| APPCOMPOSER\_CLIENT\_ID    | The name to use as Client ID when creating DAC client during the first installation.                                                                                                                 | string            | YES       |               |
| APPCOMPOSER\_CLIENT\_PWD   | The password to use as secret when creating DAC client during the first installation.                                                                                                                | string            | YES       |               |
| DB\_VENDOR                 | Database vendor. It will be use to select the SQL dialect to use                                                                                                                                     | postgres \| mssql | YES       |               |
| DB\_ADDR                   | The hostname of the database                                                                                                                                                                         | string            | YES       |               |
| DB\_PORT                   | The port of the database                                                                                                                                                                             | integer           | YES       |               |
| DB\_DATABASE               | The database name                                                                                                                                                                                    | string            | YES       |               |
| DB\_SCHEMA                 | The database schema to be used                                                                                                                                                                       | string            | YES       |               |
| DB\_USER                   | The username of the database user                                                                                                                                                                    | string            | YES       |               |
| DB\_PASSWORD               | The password of the database user                                                                                                                                                                    | string            | YES       |               |
| PROXY\_ADDRESS\_FORWARDING | ?                                                                                                                                                                                                    | true \| false     | NO?       |               |
| KEYCLOAK\_USER             | The username to assign to the Keycloak user automatically created during the first installation. It is used only for the first startup of the service                                                | string            | YES       |               |
| KEYCLOAK\_PASSWORD         | The password to assign to the Keycloak user automatically created during the first installation. It is used only for the first startup of the service                                                | string            | YES       |               |
| LOGOUT\_REDIRECT\_URI      | The URL to use after a logout. Usually it’s the URL of DAC application. i.e. “_https://myapp.decisyon.com_”                                                                                          | URL               | YES       |               |
| REDIRECT\_URI              | The URL where to redirect users after a successful login. It is the URL of the application with postfix” …/index.cas”, i.e. “[_https://myapp.decisyon.com_](https://myapp.decisyon.com)_/index.cas_” | URL               | YES       |               |
| ZONE\_ID                   | The hostname of the webapp application, i.e. “_myapp.decisyon.com_”                                                                                                                                  | string            | YES       |               |
