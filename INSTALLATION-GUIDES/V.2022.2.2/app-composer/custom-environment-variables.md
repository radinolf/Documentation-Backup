# Custom Environment Variables

It is possible to define environment variables that can be used as parameters on pages or other App Composer objects.

A custom env variables is defined using the following syntax:

`DAC_ENV_<variable_name>`

After the mandatory prefix DAC\_ENV you can set any variable name you want.

Example:

```
DAC_ENV_demo_variable=demo
```

Custom environment variables are visible also in the administrator panel in the web application.
