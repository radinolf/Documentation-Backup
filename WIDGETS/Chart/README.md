# Chart

## Overview

The **Chart** widget is a visual component that allows users to display data in a graphical format, making it easier to interpret and analyze. With the Chart widget, app developers can create a wide variety of charts such as line charts, bar charts, pie charts, and more, depending on the data they want to represent.

<figure><img src=".gitbook/assets/image (64).png" alt=""><figcaption></figcaption></figure>

Here are some key features of the Chart widget:

* **Data binding:** You may quickly and easily create a chart using a simple report as data source and bind it to the widget's property **reportAssociated.** Please note that the values used in the charts come from the **metrics** available in the report and that each items available is the **column** of the reports itself is used as additional category in the chart.&#x20;

<figure><img src=".gitbook/assets/image (12).png" alt=""><figcaption></figcaption></figure>

* **Parameters:** The chart imports and exports parameters that can be used to create interactions with other widgets in the page.
* **Customization:** app developers can customize the appearance of the chart by changing the type of the chart, colors, labels, legend, and other style properties. This allows them to create charts that match their specific needs. The widget supports several chart types.
* [**Drill-Through**](https://documentation.decisyon.com/documentation/report/visualization-and-formatting/drill-through): the widgets supports drill-through configured in the report, so that when clicking on the chart the target action configured in the report is executed.&#x20;
* **Drill down/roll up**: the widgets supports [report's grouping](https://documentation.decisyon.com/documentation/report/visualization-and-formatting/grouping-levels), so that it's possible to drill down or roll-up directly from the chart.

Chart is a powerful widget for visualizing data, providing business users with an intuitive way to explore and understand the application data.

## Widget Parameters <a href="#toc63261619" id="toc63261619"></a>

**Imported Parameters**

This widget is able to import and use any parameter available in the page.

**Exported Parameters**

Parameters may be exported to the page enabling the rpoerty **activate-params-export** (**Parameters** group) in the Chart widget. This property enables the **exp-levels**, which allows you to select which report's levels must be use to export the parameter.

<figure><img src=".gitbook/assets/image (5).png" alt=""><figcaption></figcaption></figure>

When users click on the chart, the widget will export the value of that point together with the logical names of the report levels set in the **exp-levels** property.

In the following example the exported parameter is

```
?BRAND$ZDjb1? = Microsoft
```
