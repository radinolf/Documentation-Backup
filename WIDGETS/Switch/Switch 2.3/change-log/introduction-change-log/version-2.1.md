# Version 2.1

This document section lists the New Features introduced, all the Enhancements, Know issues, Defects fixed in the released versions, and the list of the features that have been Removed/Deprecated.

### New Features  <a href="#toc68171618" id="toc68171618"></a>

This section provides a list of the new features implemented, updated to the 2.1 version.

* DC-194 – **Rename of Import value feature**

The **Import value** configuration property has been renamed to **Default Value.** Refer [here](../../whats-new/whats-new-in-this-version.md#_download_contents_from) for more details.

* DC-279 – **Pre-selection-values feature based on \<basename> parameter**

Now in the **Switch** widget the **Pre-selection** feature is based on **\<basename>** parameter. Refer here for [more](../../whats-new/whats-new-in-this-version.md#_download_contents_from) details.

* DC-285 – **Boolean parameter formats settings**

Now in the **Switch** widget, you can set the Boolean format of the main parameter value imported and exported by the widget and choose to export other value formats as additional parameters. Refer [here](../../whats-new/whats-new-in-this-version.md#_new_properties_for) for more details.

### Enhancements  <a href="#toc68171619" id="toc68171619"></a>

This section provides a list of the new improvements implemented, updated to the 2.1 version.

* DC-212– **Improved parameters export management**

The parameter export management has been improved.
