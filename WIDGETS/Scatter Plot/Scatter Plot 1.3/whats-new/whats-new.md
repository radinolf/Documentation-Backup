# What's New

## **Version 1.3**

### Flexible Layouts for Scatter Plot Widget

The **Scatter Plot** widget now works seamlessly within flex containers and the latest DAC  environment. Page developers no longer need to manually set width or height for proper display, as the widget defaults to full dimensions. This improvement ensures reliable rendering in all page layouts, enhancing dashboard consistency and reducing setup time.&#x20;
