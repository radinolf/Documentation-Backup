# Discussions with weak contextuality

The Page Developer enables Discussions that have **weak contextuality** (icon on the title bar) using the **activateDiscussions** widget property.

<figure><img src="../../.gitbook/assets/image (64).png" alt=""><figcaption></figcaption></figure>

This page property is available for the widget:

* [**Crosstab**](https://app.gitbook.com/o/-Mf1nqOzbgLBl5PsDWsN/s/rX1BuyZRABZf88qbRsCl/)
* [**Chart**](https://app.gitbook.com/s/wT0I1bAt2ozV2FstpzwR/documentation/chart)

<figure><img src="../../.gitbook/assets/image (46).png" alt=""><figcaption></figcaption></figure>

In the Web Application, the Business user click the **Discussion icon** on the title bar.
