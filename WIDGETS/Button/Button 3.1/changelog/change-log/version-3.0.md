# Version 3.0

This document section lists the New Features introduced, all the Enhancements, Know issues, Defects fixed in the released versions, and the list of the features that have been Removed/Deprecated.

### ✨ New  <a href="#toc89767737" id="toc89767737"></a>

* **DC-1083**  – **New Version Graphical Style**

The graphical style of the Button widget has changed from the previous Material Design style. Refer [here](../../whats-new/whats-new-in-this-version.md#new-version-graphical-style) for more details.

{% hint style="warning" %}
This enhancement has an impact on migrating. Please, refer to the [Migration Notes](../../migration-notes/migration-notes.md#version-3.0) section for migrating to Button 3.0
{% endhint %}

### ➕ Improved  <a href="#toc68682153" id="toc68682153"></a>

* DC-645  –  **Compatibility with Flex Container**

Upgrade for compatibility with new DAC versions. The Button now supports dynamics layouts in **Flex Container**. Refer [here](../../whats-new/whats-new-in-this-version.md#version-3.0) for more details.

* DC-816 /DC-799 / DC-1002 /DC-1023 –  **Technical Improvements**

Widget libraries and code version updates.

### ⛔ Removal

* DC-1069 – **Removal of zoom property**

The `zoom` property is no longer available in the Button widget. Buttons previously using zoom options Medium, Large, and XLarge will default to "Normal".&#x20;

### DAC Compatibility <a href="#toc57372503" id="toc57372503"></a>

This section provides information about compatibility requirements between the widget and the versions of DAC.

{% hint style="success" %}
It is possible to install the **Button 3.0** in **DAC version 2022.2.0 and later.**
{% endhint %}
