# Change Log

The Change Log' is a document distributed together with the release of a new version (minor release or patch) of the Widget and is prepared as a log of solved bugs and of the functions entered in the Platform; in fact the 'Change Log' lists the changes and improvements made to the widget and is distributed only after the widget has been carefully tested and approved based on the specifications provided by the development team.
