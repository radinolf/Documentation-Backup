# Version 1.1

This document section lists the New Features introduced, all the Enhancements, Know issues, Defects fixed in the released versions, and the list of the features that have been Removed/Deprecated.

### New Features  <a href="#toc57372503" id="toc57372503"></a>

This section provides a list of the new features implemented, updated to the 1.1 version.

* DC71 – **Timeline min-height**

Reduced the min-height of the widget

### Enhancements  <a href="#toc57372504" id="toc57372504"></a>

This section provides a list of the new improvements implemented, updated to the 1.1 version.

* DC69 – **Use different colors in the timeline**

Introduced the possibility to assign different colors to the events in the timeline. Refer [here](../../whats-new/whats-new.md#_toc57372501-1) for more details.

* DC77 – **Show only the date in the timeline labels**

The widget introduced the new configuration property **Date/time Format viz** to set the date/time format that also provides options to show the date, without hour and minutes, in the timeline labels.
