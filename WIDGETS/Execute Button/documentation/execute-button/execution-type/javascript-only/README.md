# JAVASCRIPT ONLY

## Overview

This action allows you to execute JavaScripts code set in the **javaScript-code-after** property. It is possible to use page parameters in the code or to create new parameters in the page from the code. In the configuration dialog on page designer it's possible to read all the [Javascript API](https://documentation.decisyon.com/documentation/page-designer/api-javascript) exposed by DAC that is possible to use in the execute button widget.&#x20;
