# CODE SNIPPET

## Overview

This option allows the execution of a **code snippet** object previously defined in the **Code Snippet Editor** (**Application --> Code Snippet Editor**). Then, by selecting **CODE SNIPPET** in the **execution-type** property, the **code snippet** property is available for associating the code snippet.

Code snippet associated to the execute button widgets are capable to import parameters from the page and use their value in the business logic coded in the Java program.&#x20;
