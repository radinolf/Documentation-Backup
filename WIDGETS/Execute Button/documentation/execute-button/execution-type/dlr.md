# DLR

## Overview

With this execution type it is possible to execute a [DLR](https://documentation.decisyon.com/documentation/app-functionality/data-load-via-report-dlr) object previously defined using the **Data Load via Report (DLR)** (**Data Model-->Data Load via Report (DLR**). Then, by selecting DLR in the **execution-type** property, the **dlr\*** property is available to associate the object.
