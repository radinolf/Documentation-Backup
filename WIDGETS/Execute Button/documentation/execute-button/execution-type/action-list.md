# ACTION LIST

## Overview

**Action List** is an object of App Composer that can be used to execute sequentially a list of other objects, called **Actions**.&#x20;

This operation allows you to execute an action list defined in **Action List** (**Administration -->Action List** from the Side Navigation Menu of the web application).
